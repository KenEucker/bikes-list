Documentation for the Shift Calendar

This directory contains documentation for the Shift calendar. The documentation is intended to help you adapt the calendar software to your needs.

It is not intended for end-users. The calendar is meant to be intuitive enough that very little documentaion is needed to actually use it. The only user documentation resides in the explain directory.

The documentation is split into multiple files:

features.html
Brief list of some of the calendar's distinctive features.
install.html
Step-by-step instructions on how to get started with the calendar.
appearance.html
This describes how to change the colors of the pages. Sadly, the calendar doesn't yet support "themes" so this is harder than it should be.
code.html
This lists the source files and directorie, and explains what they're for and how they relate to each other.
database.html
Details about the SQL tables used by the calendar.
update.html
Technical description of how dates are updated.
bugs.html
Yeah, it ain't perfect.
future.html
This lists some changes I'd like to make to the calendar some day.
